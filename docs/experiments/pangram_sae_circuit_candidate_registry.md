# Pangram SAE Circuit Candidate Registry

Date: 2026-06-23

This registry summarizes the current circuit candidates after the first seed, peak-token, intervention, and control passes. It is a working queue, not a final mechanistic claim.

## Evidence Levels

- `seed`: prompt-overlap and semantic evidence only.
- `intervention`: upstream ablation changes downstream latent activity.
- `controlled`: matched controls show the downstream change is specific to the selected pair.
- `held-out`: pattern-selected documents outside the latent-selected target export reproduce the downstream effect.
- `counterfactual`: local text edits in the same documents reduce the expected downstream activation.
- `score-path`: upstream/downstream/joint ablations move the detector target score in the expected direction.
- `score-only`: upstream ablation moves the detector score, but the nominated downstream latent does not move enough to support a circuit edge.
- `dose-response`: controlled feature-density sweep changes latent activation in the expected direction.
- `token-locus`: token-level peaks identify what local tokens carry the latent activation.
- `blocked`: useful but missing enough shared prompts or not yet tested with the available layer-20 checkpoint.
